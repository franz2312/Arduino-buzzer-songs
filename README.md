# Arduino-songs
you could use arduino uno/nano and buzzer to play songs!

This code made by franz2312

All the melodies were compiled from https://github.com/hibit-dev/buzzer.

The code still has a few bugs, some of which are:

--------
THE BUGS
--------

To change songs, you first have to press the stop button, then the change song button.

Sometimes the buttons are not recognized.

---------------
Connection pins:
----------------

D2: Stop/Start button

D3:
Change music button

D9: Buzzer
